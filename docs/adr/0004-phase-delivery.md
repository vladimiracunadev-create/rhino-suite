# ADR 0004 — Entregas por fases

**Estado:** aceptada

## Decisión

El proyecto evoluciona en un único repositorio. Cada fase debe dejar una base ejecutable, probada y documentada antes de iniciar la siguiente.

## Consecuencias

- No se desarrollan cuatro editores incompletos en paralelo.
- El núcleo y los contratos se estabilizan primero.
- Cada ZIP representa una versión evolutiva del mismo producto.
